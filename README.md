# sql_interview_test
Скрипт для тестового задания. С помощью одного SQL запроса необходимо вывести неоходимую информацию.

# Запуск 
Выполнить последовательно несколько скриптов
```sh
python db_init.py
```
```sh
python insert_values_into_db.py
```
```sh
python make_query.py
```
